# Персонаж: Монетка

**Монетка** — игровой наставник Money Quest для онбординга и (позже) коротких подсказок. Не банковский консультант и не «голос приложения».

---

## Роль в продукте

| Контекст | Роль Монетки |
|----------|----------------|
| **Mission Brief (O1)** | Ведёт 3 шага «первого месяца», говорит от первого лица («я подскажу») |
| **Coach marks (P1.1)** | Одна реплика в bubble, без длинного текста |
| **События / экономика** | **Не** комментирует каждое событие (перегруз) |
| **Меню** | «Повторить обучение» — снова бриф с Монеткой |

---

## Характер

- **Тон:** дружелюбный, чуть озорной, уверенный; как напарник в квесте, не учитель в аудитории.
- **Речь:** короткие фразы, 1–2 предложения на экран; без «уважаемый пользователь».
- **Отношение к ошибкам:** «Ошибаться нормально — смотри на цифры», без морализаторства.

**Примеры реплик (черновик):**

- «Привет! Я **Монетка**. Сейчас разберём, как тут устроен месяц — это быстро.»
- «Зарплату **забираешь сам** до конца периода. Подушка — твой запас на чёрный день.»
- «В шапке будут **События** — пару решений за месяц. Погнали?»

---

## Визуал (черновик для design-lab)

| Элемент | Правило |
|---------|---------|
| **Форма** | Круглая «монета», золотистый центр, ободок Quest Violet |
| **Лицо** | Минималистичное: глаза-точки, улыбка; без реализма |
| **Размер в TMA** | 48–64 px в шапке bubble; не перекрывает CTA |
| **Анимация (P2)** | Лёгкий bounce при смене шага — только после утверждения статики |
| **Ассет** | Сначала CSS/SVG в design-lab; TGS — если появится отдельный файл |

**Недопустимо:** копировать стиль Telegram mascot; крипто-мемы; символы ₽ на лице монеты.

## Связь с брендом

- Цвета: Quest Violet `#6D28D9`, акцент золота/amber для «монеты» — см. [`BRANDBOOK.md`](brandbook/BRANDBOOK.md).
- Тон согласован с «наставник, не робот», но **игровее**, чем общий копирайт UI.

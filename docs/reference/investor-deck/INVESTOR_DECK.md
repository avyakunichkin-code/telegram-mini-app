# Money Quest — Investor UI Deck (v1)

Этот документ — **эталонный дизайн всех ключевых экранов** Money Quest (Telegram Mini App) для демонстрации инвесторам.

## Как посмотреть / как сделать PDF

- Открой `docs/reference/investor-deck/deck.html` в браузере.
- Экспорт в PDF: **Print → Save as PDF** (формат A4).
- Рекомендации печати: поля по умолчанию, фоновые изображения включить (если браузер предлагает галочку).

## Состав экранов (что внутри deck.html)

- **Старт / Сохранения**: быстрый вход в активный профиль + KPI + 1 главный CTA.
- **Логин / Регистрация**: единый визуальный язык (hero + карточка-форма), без “телеграм-синевы”.
- **Мастер: Режим и шаблон** — выбор **Game** / **Plan**; для Game — слот шаблона / сложность (в коде пока может отображаться legacy «сложность light/hardcore» — см. `CLAUDE.md`).
- **Мастер: Базовые параметры** — для Plan: наличка + зарплата + активы + обязательства (компактно); для Game — урезанный или скрытый шаг при старте с шаблона.
- **Игра: Главная (Dashboard)**: состояние периода + 2 KPI + 2 действия.
- **Игра: Финансы**: сетка табов 2×2, сценарные карточки.
- **Игра: Аналитика**: цель + динамика + риски (3 вопроса → 3 карточки).
- **Игра: События**: карточка решения + последствия (фокус на выборе).
- **Игра: Меню**: 3 действия сессии + опционально “о проекте”.

## Дизайн-принципы (для плотной информации)

- **1 экран = 1 главный вопрос** (остальное — подписи/вторичный уровень).
- **Не больше 2 “больших” чисел** на видимой части экрана одновременно.
- **Группировка через карточки**: заголовок → 1–2 строки пояснения → строки/плашки → CTA.
- **Акцент**: Quest Violet для primary/active, Emerald для “в плюс”, Red для рисков.

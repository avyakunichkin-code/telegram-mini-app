1. Логотип — варианты на фонах

Светлый фон • основной знак

Тёмный фон • монохром

Минимальные размеры и зона охранения

  • Минимальная высота монограммы в TMA header: 24 px
  • Отступ свободного поля вокруг знака: ≥ 1/2 высоты монограммы
  • Печать визитки: монограмма не менее 8 mm
Нельзя: растягивать по вертикали/горизонтали без пропорций; класть на «шумный» фото без затемняющей подложки; добавлять символ валют внутрь знака (₽ € $) без отдельного макета; использовать несколько несогласованных градиентов на маленьких размерах.

3. Типографика

СтильРазмерВесПрименение
H1 веб28–32 pxSemiboldГерои лендинга
H222–24 pxSemiboldСекции
Body15–17 pxRegularОперационный текст
Caption12–13 pxRegularПодписи, disclaimers

Верно: слева по краю в карточках продуктовых экранов; табличные числа для балансов; один «главный» акцент весом на экран.

Неверно: заголовки только CAPS без причины; слишком плотный интерлиньяж; много разных семейств на одном экране.

5. Паттерны и иконсет

  • Паттерн «Сетка периода»: шаг 8 px, цвет линии #E5E7EB с opacity 0.25–0.35, не под обычный body-текст.
  • Иконография: outline 24 px, stroke ~1.75 px, округлые края; один стиль во всём продукте.

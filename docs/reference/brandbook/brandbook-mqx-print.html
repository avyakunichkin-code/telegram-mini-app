5. Финансы (капитал)

Порядок: Доходы → Расходы → Инвестиции → Страховки → Имущество → Обязательства.

Сегмент «Добавить | Мои (N)» — внутри mqx-capital-card, не в hero.

Строки: MqxFinListRow · каталог: CapitalPositionCard.

Правила: один violet hero на вкладку · tg-theme-* + glass, не белый кусок в тёмном TG · модалки с последствием · prefers-reduced-motion.

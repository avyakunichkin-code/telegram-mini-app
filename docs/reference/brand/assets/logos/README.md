# Логотипы «ТВОЙ ХОД» — экспорт для TG, сайта, соцсетей

**Формат:** SVG, фон прозрачный.  
**Цвета:** ТВОЙ — Coin Gold `#EAB308`, ХОД — Quest Violet `#6D28D9`, обводка `#FFFFFF` (читаемость на тёмном).

## Нужна ли подпись «Финансы как игра»?

| Носитель | Рекомендация |
|----------|----------------|
| Аватар группы / бота TG | **Без** tagline — только L4 или L2 |
| Шапка сайта, обложка канала | **С** tagline — L3 или L3h |
| Посты, сторис, наложение на фото | **Без** tagline — L1 или L2 |
| Лендинг, презентация | **С** tagline — L3 |

По канону маркетинга tagline **обязателен в коммуникациях**, но не обязан быть **внутри** одного графического файла — можно строкой под логотипом.

## Варианты

| ID | Файл | Назначение |
|----|------|------------|
| **L1** | `L1-horizontal.svg` | Одна строка, без tagline — хедер, водяные знаки |
| **L2** | `L2-stacked.svg` | Две строки, без tagline — узкие блоки |
| **L3** | `L3-stacked-tagline.svg` | Столбик + «Финансы как игра» — сайт, описание канала |
| **L3h** | `L3-horizontal-tagline.svg` | Строка + tagline снизу — широкие баннеры |
| **L4** | `L4-monogram-avatar.svg` | Монограмма **ТХ** — аватар TG (квадрат 512×512) |

## Экспорт в PNG

1. Откройте [`preview.html`](./preview.html) в браузере.
2. Правый клик по варианту → «Сохранить изображение» **или**
3. Figma / Inkscape: импорт SVG → Export PNG @2x/@3x.
4. Для аватара TG: L4 → **512×512** PNG.

## Тёмный фон

На очень тёмном фоне белая обводка уже встроена. Если кладёте на **светлый** фон без обводки — напишите, сделаем отдельный набор `*-on-light.svg` (без stroke, только заливка).

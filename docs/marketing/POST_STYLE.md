# Стиль постов (черновик для согласования)

Документ **уточняется** при первых утверждённых постах. До статуса `approved` агент предлагает варианты и фиксирует ваш выбор здесь.

## Голос

| Принцип | Как в постах |
|---------|----------------|
| Наставник, не банк | Простые слова, без «обязательств по кредитному договору» |
| Игра и прогресс | «В Money Quest теперь…», «попробуйте в новой партии» |
| Честность | Не обещать релиз-дату и фичи, которых нет в билде |
| Монетка (опционально) | От первого лица ж.р., лёгкий задор — как в [`CHARACTER_MONETKA.md`](../reference/CHARACTER_MONETKA.md) |

**Избегаем:** крипто-мемы, паника, внутренние номера миграций, длинные списки коммитов.

## Структура `post.md`

```markdown
# Платформа: Telegram (канал)

## Заголовок
…

## Тело
…

## CTA
…

## Хештеги
#MoneyQuest …
```

Длина Telegram: ориентир **до ~900 символов** в одном сообщении (с запасом под подпись к фото).

## Визуал (картинка)

- Персонаж **Монетка** на каждом посте (PNG, прозрачный или тёмный фон по брендбуку).
- Палитра: см. [`BRANDBOOK.md`](../reference/brandbook/BRANDBOOK.md) — тёмный фон, акценты violet/gold.
- На изображении — **крупный** короткий заголовок (3–5 слов) или без текста; мелкий текст не использовать.
- Логотип MQ — только если читаемо в мобильной ленте.

## Платформы (заготовка)

| Платформа | Особенности |
|-----------|-------------|
| Telegram | Картинка + подпись; можно без заголовка |
| VK | Чуть длиннее допустимо; те же ассеты |
| Другое | Уточнять при запросе |

# Лестница баланса Game-шаблонов (4 ранга)

**Статус:** актуально после выноса жилья из burn (май 2026).  
Код: `backend/main.py`, `expense_template_defaults.py`, миграции `0029`, `0030`.

## Принципы

1. **Free flow** = `зарплата − burn (без жилья) − обслуживание активов − проценты по долгам`.
2. **Жильё** — не в burn: **аренда** (`leased_dwelling`), **содержание своей** (`home` / `house` / …), **ипотека** в liabilities.
3. **Доходной недвижимости в старте нет** ни у одного шаблона.
4. Зарплата по рангу: **62,5 → 100 → 150 → 200** тыс. (Студент 62,5k).

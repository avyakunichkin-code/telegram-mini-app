# Баланс XP: вечерняя сессия Game Mode

Зафиксировано после idea-refine (май 2026): тон продукта, кривая уровней, ритм периода, достижения.

## Problem Statement

**Как сделать прогрессию ощутимой в первые 5–6 минут (до 4-го уровня), удержать вечернюю партию 40–60 минут и оставить «фантомный» 12-й уровень как финальный трофей без обязательных инвестиций для победы на basic?**

## Продуктовые решения

| Тема | Решение |
|------|---------|
| Тон | Игра с эмоциями и достижениями, не учебник и не hardcore-симулятор |
| Длина basic | ~40–60 мин, медиана победы период **10–12** |
| Инвестиции на basic | Победа без них возможна, но **затруднена** |
| Макс. уровень | **12**; накопительно **2500 XP**; переход 11→12 — **115 XP** («фантомный» финиш) |

## Кривая уровней (канон v2)

Код: `XP_NEED_BY_LEVEL` в `backend/app/game_rules.py`, `CHARACTER_MAX_LEVEL = 12`.

## Ритм периода (единый пакет `process_period_end`)

| Компонент | XP |
|-----------|-----|
| `period_close_base` | 12 |
| `period_close_salary` | +10 если зарплата за период |
| `period_close_savings` | +0…20 (`contrib / 2000`, cap 20) |
| `claim_salary` (мгновенно) | 0 (бонус в close) |
| `safety_contribute` | 3, max **2** раза за период |
| `safety_withdraw` | 1, max **1** раз за период |

## Достижения (xp_reward по tier)

База **26**, множители по цепочкам: safety/credit **×1.08**, deposit/insurance/capital/invest **×0.92–1.0**.

| tier | ~XP (base 26) |
|------|----------------|
| 1 | 26–28 |
| 2 | 62–66 |
| 3 | 104–110 |
| 4 | 152–160 |

# Позитивный выбор сценария на старте (idea-refine)

**Статус:** черновик направления (май 2026)  
**Связано:** `GameTemplatePickScreen`, `starter_template_presentation.py`, MQX T2 strips, [лестница баланса](starter-template-balance-ladder.md)

## Problem Statement (HMW)

**Как помочь игроку выбрать сценарий так, чтобы он чувствовал азарт и компетентность, а не страх ошибки и иерархию «семья = сложно»?**

## Recommended Direction

1. **Архетип вместо абстракции** — название = роль игрока (`Студент` → `Профессионал` → `Руководитель` → `Предприниматель`), не «базовый старт» / «красная зона».
2. **Бейдж = энергия, не оценка** — цвет сохраняем; подписи: `Старт` / `Ритм` / `Эксперт` / `Драйв` (не Легко/Сложно/Экстрим).
3. **Bullets = факты** — «Доход ~N ₽/мес», «Расходы ~N ₽/мес»; без «скромная/средняя» при разных цифрах в линейке.
4. **Compare = следующий уровень крутости** — «уже интереснее», «тренируешь», «твой драйв»; без «сложнее базового», «давление», «пустой счёт» как угрозы.
5. **Сложность экономики ≠ мораль** — не связывать rank с «один/семья»; ипотека = «свой дом», не «семейные ценности».

## Key Assumptions

- Игрок 18–35 в TMA хочет **быстро понять отличие** сценариев (3–4 bullets + одна compare-строка).
- **difficulty_rank** по-прежнему отражает экономическую нагрузку в симуляции; UI только переименовывает восприятие.
- Четыре шаблона MVP достаточно; пятый+ — отдельная итерация каталога.

- [x] Нейтральная генерация highlights без противоречивых adj.

## Not Doing (пока)

- Скрывать цифры зарплаты «чтобы не пугать» — цифры честность; пугает оценочный язык.
- Пятый шаблон «Семья» с негативным rank.
- Пересортировка salary между шаблонами только ради монотонных подписей.

## Варианты названий (divergent → clusters)

### Кластер A — карьерная лестница (выбрано в prod)

| rank | title | бейдж |
|------|--------|--------|
| 1 | Студент | Старт |
| 2 | Профессионал | Ритм |
| 3 | Руководитель | Эксперт |
| 4 | Предприниматель | Драйв |

### Кластер B — жизненные главы (позитив, без office-speak)

| rank | вариант | подзаголовок (идея) |
|------|---------|---------------------|
| 1 | Первый кошелёк | «Учишься жить по периодам» |
| 2 | Свои колёса | «Авто и кредит — первый апгрейд» |
| 3 | Свой адрес | «Дом + платежи — новый уровень» |
| 4 | На нескольких фронтах | «Два долга — много решений» |

### Кластер C — игровой тон (мотивация)

| rank | title | бейдж |
|------|--------|--------|
| 1 | Разминка | Старт |
| 2 | Квест | Ритм |
| 3 | Босс-раунд | Эксперт |
| 4 | Хардкор* | Драйв |

\* слово «хардкор» не в UI — только «Драйв».

### Кластер D — aspirational (ещё круче для A/B)

- **Фрилансер** (между 2–3): нестабильный доход, один долг — будущий шаблон.
- **Инвестор в себе** (rank 2 alt): кредит на обучение/авто как «вложение».
- **Создатель бизнеса** (rank 4 alt): вместо «Предприниматель» — меньше ассоциации с «долгами».

### Inversion / stress-test

- **Убрать rank-бейдж совсем** — только цвет полоски: риск «не понять градацию».
- **Показать только «рекомендуем для первой игры»** на rank 1 — остальные без негатива, с «для опытных» → заменить на «для любителей драйва» на rank 4.

## Позитивный настрой: практики копирайта

| Вместо | Использовать |
|--------|----------------|
| Скромная / средняя зарплата | Доход ~N ₽/мес |
| Сложнее предыдущего | Уже интереснее / следующий уровень |
| Давление, красная зона | Плотный ритм, много решений |
| Пустой счёт | На счёте ~N ₽ (факт) |
| Для опытных | Для любителей драйва |
| Ошибки по кэшу | Учишься вести кэш |

**Монетка (идеи реплик):** «Выбери, в какой роли хочешь прокачать деньги — от первого бюджета до полного драйва», не «осторожно, ипотека тяжёлая».

## Дополнительные шаблоны (бэклог)

1. **Фрилансер** — доход 55k, нерегулярные события (когда появятся).
2. **Релокация** — высокие расходы, нулевые долги, цель подушка 6×.
3. **Сберегатель** — высокий cash, низкая salary, без кредитов — rank 1 alt для «люблю копить».

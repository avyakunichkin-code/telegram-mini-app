# Achievements & Level — раскрываемый блок + монетки

Запуск: `cd design-lab/achievements-progress && npx serve .`

**Spec:** [`docs/specs/features/SPEC_achievements.md`](../../docs/specs/features/SPEC_achievements.md) §11.

**Пайплайн:** варианты здесь → утверждение → `mqx/` + `#/dev/mqx` → `DashboardPremium` / `MenuPremium`.

---

## Проблема

- Главная **компактная**, но игроку нужно видеть **рост** и **последние вехи** без 5-го таба.
- Достижения уже в API; в prod — только тосты и модалка закрытия периода.
- Нужны **различимые** маленькие монетки (как метрики ~20–24px), не перегружая flat-стек D′.

---

## Варианты — блок «Уровень» (L)

| ID | Идея | Свёрнуто | Развёрнуто |
|----|------|----------|-----------|
| **L0** | Текущий prod (`MqxLevelBlock` flat) | Уровень + XP + hint + bars | — (референс) |
| **L1** | Collapsible card, chevron справа в заголовке | Как L0 + ▼ | + strip монеток + ссылка |
| **L2** | L1 без bars в развёрнутом (только XP + монеты) | Минимум высоты | Полоса 5 монет + «Все →» |
| **L3** | L1 + счётчик «Достижения 4/24» в заголовке | Уровень + `4/24` + meter | Strip + link |
| **L4** | Accordion внутри `mqx-card` с тонкой границей | Chip «Ур.3» + meter | Полный блок как отдельная секция |

**Тестовые данные (все варианты):** Ур.3, 120/200 XP, hint «Ещё 80 XP до 4 уровня», 5 монет: бронза→серебро→золото→платина→звезда (последняя — tier 4 capital).

## Варианты — монетки (C)

| ID | Идея |
|----|------|
| **C1** | Плоский круг + CSS gradient по металлу (бронза/сер/зол/плат) |
| **C2** | SVG с внутренним кольцом и бликом (читаемость 20px) |
| **C3** | Tier 4 = **звезда** вместо круга; tier 1–3 = монеты |
| **C4** | Монета + микро-иконка категории цепочки (подушка/вклад…) в центре |
| **C5** | Locked = серый контур; unlocked = металл + лёгкая тень |

Рекомендация для сравнения: **C2 + C3** (звезда только на tier 4).

## Варианты — страница «Все достижения» (P)

| ID | Идея |
|----|------|
| **P1** | Вертикальный список цепочек; tier = строка [монета][текст][XP] |
| **P2** | Сетка 4 монеты на цепочку + описание под рядом |
| **P3** | Full-screen overlay поверх Game (шапка «Назад»), таббар виден |
| **P4** | Отдельная «вкладка» без таббара + только Back (как Event overlay) |

Spec допускает **P3** (таббар остаётся) или hash-route — выбрать один при утверждении.

---

## Критерии выбора

1. Свёрнутый блок **не выше** ~72px (без bars).
2. Монеты различимы на **320px** без zoom.
3. «Все достижения →» заметна, но не спорит с CTA зарплаты/периода.
4. Quest Violet только для ссылок/акцентов, не для золота монет.
5. Тёмная тема: металлы не сливаются с `mqx-glass` фоном.

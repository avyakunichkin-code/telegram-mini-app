# Invest forms — design lab

Витрина форм **депозита** и **облигаций**: подчёркнутый ввод, ползунок 0…баланс, фиксированная ставка.

## Открыть

```bash
cd design-lab/invest-forms
npx serve .
```

## Варианты

| ID | Название | Идея |
|----|----------|------|
| **A** | Минимал | Подчёркивание + ползунок, ставка строкой справа |
| **B** | Полоса + форма | Accent + chip ставки |
| **D ★** | Компактная строка | Сумма + chip (14% / 16% годовых) — **в prod** |
| **C** | Hero-сумма | Крупные цифры, ползунок ниже, ставка под заголовком |
| **D** | Компактная строка | Сумма и ставка в одной линии, узкий ползунок |
| **E** | Плотный TMA | Минимум вертикали: сумма + ползунок + chip ставки внизу |

**Prod:** `InvestProductForm` + `InvestAmountControl` в `frontend-react/src/components/`.

Ставки MVP: депозит **14%**, облигации **16%** (`constants/investProducts.js`). Облигации доходнее депозита (больше риск/волатильность в реальности — в MVP упрощённо).

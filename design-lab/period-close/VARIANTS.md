# Итог периода — варианты (design-lab)

Замена огромного `PeriodCloseModal` (telegram-ui `Modal` + длинный breakdown).

## Механика (утверждаем вместе с визуалом)

| Условие | Поведение |
|---------|-----------|
| `period_index` закрытого месяца **≤ 3** | Итог **всплывает автоматически** после `POST /api/game/time/next` |
| **≥ 4** | Авто-всплытие **выключено**; внизу дашборда **«хвостик»** — тап открывает итог **прошлого** периода |
| Закрыли итог | Данные остаются в `lastPeriodClose` до следующего закрытия; хвостик виден, пока есть что показать |

Хвостик: подпись вида **«Период 3 · итоги»**, не перекрывает таббар.

## Содержимое (краткая выжимка)

Не полный breakdown по строкам, а **дельты** с привычными глифами (`coin` / `up` / `down` / `term`):

| Метрика | Смысл |
|---------|--------|
| Баланс | Δ cash за период (или «стало» vs «было») |
| Доходы | Сумма поступлений (зарплата, активы, купоны…) |
| Расходы | Сумма списаний (жизнь, обязательства, обслуживание…) |
| Подушка | Δ `safety_fund_balance` |
| Инвестиции | Δ депозиты + облигации (капитализация / взносы) |

Опционально одна строка **+XP** и бейдж достижения (если есть), без отдельных секций на полэкрана.

### Задел API (prod, после утверждения UI)

Расширить `PeriodCloseSummary` в `backend/app/schemas.py`:

- `closed_period_index: int`
- `cash_before`, `cash_after` (или `cash_delta`)
- `income_total`, `expense_total`
- `safety_fund_delta`, `invest_capital_delta`
- `breakdown` — оставить для детализации / аналитики, не показывать в компактном UI

Считать в `process_period_end` / `_period_close_summary` из транзакций периода и `PeriodEconomyClosing` (два последних closings).

## Варианты

| ID | Название | Идея |
|----|----------|------|
| **A** | Compact sheet | Нижний лист ~38% экрана, затемнение, 5 метрик в сетке |
| **B** | Hero strip | Полоска под таймером (не modal), «свернуть» |
| **C** | Chips 2×2 | Четыре крупных чипа + подушка/инвест отдельной строкой |
| **D** | Tail tab ★ | Хвостик + тот же контент что A, но открывается по тапу с периода 4 |
| **E** | Float card | Компактная карточка над таббаром, без fullscreen scrim |

Рекомендация для TMA: **D** (механика хвостика) + тело **A** или **C**.

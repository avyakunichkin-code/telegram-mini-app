# Утверждённая раскладка дашборда (main, без hero)

Статус: **утверждено** для design-lab → MQX → `DashboardPremium`.

## Порядок блоков (сверху вниз)

```
┌─────────────────────────────────────┐
│ ① УРОВЕНЬ (всегда виден)            │  ← бывший «Герой» + «Фин. стратег»
│    без kicker «Герой»               │     один компактный блок, sky-стиль
│    заголовок: «Уровень N»           │
│    XP + очки + bars (сжато)         │
├─────────────────────────────────────┤
│ ② СВОРАЧИВАЕМЫЙ «Дашборд периода»    │  ← клик по панели
│    свёрнуто: сводка целей + 4 chip  │     (цели + финансы)
│    развёрнуто: цели + сетка финансов │
├─────────────────────────────────────┤
│ ③ ДЕЙСТВИЯ ПЕРИОДА (всегда виден)    │  ← Зарплата = primary
└─────────────────────────────────────┘
```

## Что куда

| Блок | Сворачивается? | Содержимое |
|------|----------------|------------|
| **Уровень** | Нет | `Уровень N`, XP-прогресс, очки геймификации, 2–3 bar дохода/долгов (из «стратега»). Без слова «Герой». |
| **Дашборд периода** | Да | **Свёрнуто:** «2 из 3 целей · Почти» + chip: баланс, подушка, поток, расходы. **Развёрнуто:** полный список целей победы + блок «Финансы» (2×2 + «Детали»). |
| **Действия периода** | Нет | 4 кнопки; **Зарплата** — `primary`. |

## Что убрали с дашборда

- Отдельная большая карточка «Герой».
- Отдельная карточка «Финансовый стратег» внизу (влита в «Уровень»).
- Стрик на главном экране → **Расходы** (в chip финансов).
- Цели и финансы вне сворачиваемой панели.

## Компактный «Уровень» — варианты в lab

| ID | Идея |
|----|------|
| **H1** | Одна строка: заголовок + XP-bar inline + очки |
| **H2** | Узкая sky-карточка: шапка + bar + 2 мини-bars |
| **H3** | Три chip в ряд (ур./XP, очки, доход) |

См. секции **H1–H3** и полный макет **D′** в `index.html`.

## Не в scope

- Hero (таймер, play, pills) — спринт `dashboard/`.
- Аналитика как отдельный экран.

# Z-NEEDS v7 — плоский блок + действия

**Концепт (общий для всех вариантов):**

1. **`h2.mqx-finance-static__title`** вынесен из карточки — как «Финансы периода» / «Действия периода».
2. **Всегда 4 шкалы** справа от портрета; accordion и chevron **сняты**.
3. **Нет summary** («Есть просадка», «Истощение» в шапке) — статус только в подписи справа от бара.
4. Bleed **risk** остаётся при `streak > 0`.

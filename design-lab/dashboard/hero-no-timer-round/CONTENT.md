# Hero без таймера — контент

## Продуктовое видение

- **Период** = игровой месяц, который **открыт**, пока игрок сам не закроет.
- **Главное действие** на дашборде — **закрыть месяц** (primary), не «дождаться времени».
- **События** — вторичный pill, не конкурирует с CTA по весу.

## Копирайт (черновик)

| Элемент | Текст |
|---------|--------|
| Статус | «Месяц открыт» |
| Подпись | «Закрой, когда сделаешь дела» |
| Primary CTA | **«Закрыть месяц»** ★ FINAL |
| Secondary | «События» (+ badge), под primary в колонке справа |
| Период | «Период» · **#3** |
| Layout | **H2** — кнопки справа ★ FINAL |
| План месяца (чипы) | **TB1.1** — см. вариант H3 в lab |

## Онбординг (связь)

- Шаг 1: номер периода + primary CTA, без ▶/⏸ и MM:SS.
- Шаг 4: нажать «Закрыть месяц» (practice).

## Не показываем

- MM:SS, % прогресса времени, play/pause, «Прогресс месяца» как таймер.

# Dashboard — визуальные скины (L3)

**Открытый раунд:** [`period-actions-round/VARIANTS.md`](period-actions-round/VARIANTS.md) — chip-действия, Монетка справа, подписи.

Порядок блоков везде **L3**: финансы → цель → действия → уровень.

| ID | Название | Идея |
|----|----------|------|
| **S1** | Flat | Без рамок: chip — лёгкий тинт, разделители — отступы, Монетка и ссылка без обводок |
| **S2** | Air | Воздух: метрики без «коробок», иконка + текст в ряд, минимум линий |
| **S3** | Soft | Мягкие панели: тень вместо border, единый «лист» контента |
| **S4** | Strip | Кнопки без рамок; финансы-лента (не берём сетку) |
| **S5** ★ | **Unified** | **2×2** финансы + inset-линии + единая страница (hero / градиент / белый таббар) + уровень градиентом |

## Общие правки (все скины)

- **XP** — голубой градиент (`#38bdf8` → `#818cf8`), не зелёный.
- **Шеврон** — SVG-иконка, без фона.
- **×** и **«Все финансы →»** — без рамок/кнопочного фона.
- **Уровень раскрыт** — шкала XP остаётся видимой (`xp-pinned`).
- **Длинные суммы** — уменьшение шрифта, без переноса (кнопка «Длинные суммы» в lab).

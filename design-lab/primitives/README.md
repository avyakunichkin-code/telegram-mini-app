# Примитивы MQX — design lab

Кнопки, пилюли (hero), чипы (период / XP), прогресс-бар.

## Просмотр

```bash
cd design-lab/primitives
npx serve .
```

## Утверждённый гибрид (D ★ + C + B/A)

| Слой | Источник | Параметры |
|------|----------|-----------|
| **Размеры** | D | Кнопки `9×12`, radius `12`; пилюли `8×11`, radius `10`; период `7×10`; прогресс **6px** |
| **Шрифты** | C | Кнопки **15px / 700**; пилюли **13px / 700**; XP-чип **12px / 700**; период: label 10px caps, value **16px** |
| **Прогресс цели** | D + B + A | Трек 6px (D); fill **emerald-градиент** `#059669 → #34d399 → #2dd4bf` |
| **Прогресс XP** | D + A | Трек 6px; fill **sky-градиент** `#38bdf8 → #818cf8` |
| **XP-чип** | outline | Белый фон, обводка и текст `#818cf8` (тёмный конец градиента XP) |
| **Подписи** | D (уточнение) | Полные: «Период» + цифра, «События» + badge — без «П.3» |

В prod: `mqx/primitives/*`, стили в `frontend-react/src/index.css`, каталог `#/dev/mqx`.

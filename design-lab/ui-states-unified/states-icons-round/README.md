# states-icons-round

HTML-витрина **B1 + B2 + B3**: empty / error / loading · ritual icons · chips 2×2.

## Запуск

```powershell
cd design-lab/ui-states-unified/states-icons-round
.\sync-lab.sh
npx serve .
```

Открыть `http://localhost:3000/` (или порт serve). Переключить **тёмную** тему.

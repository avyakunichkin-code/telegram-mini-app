## S1 — Empty (список / раздел капитала)

| ID | Идея | Плюсы | Минусы |
|----|------|-------|--------|
| **A** | Текущий **`mqx-capital-empty`**: текст по центру, опциональный violet CTA | Уже в prod, спокойный | Без иконки, слабая «игровость» |
| **B** | A + **иконка 32px** (shield / folder / coins) в `mqx-accent--*` круге | Узнаваемость без Монетки | Ещё один glyph в наборе |
| **C** | **Inline** в карточке: полоска + текст слева (как строка, не billboard) | Меньше вертикали | Может сливаться с `MqxFinListRow` |
| **D** | **Монетка 48px** + заголовок + подпись + CTA (как coach-lite) | Эмоция, бренд | Тяжело для каждого пустого списка |

**Рекомендация для выбора:** B или C для списков; D — только для «первого пустого» (нет ни одной позиции в игре), если нужен wow.

## S2 — Error (секция не загрузилась)

| ID | Идея | Плюсы | Минусы |
|----|------|-------|--------|
| **A** | Только **toast** (как сейчас) — в lab показать как референс «не primary» | Минимум UI | Не помогает, если список пуст из-за ошибки |

**Не утверждать A** как единственный паттерн для списков с fetch.

## S4 — Icons: итог периода (ritual beats)

Текущий prod: символы `+`, `!`, `◎` в `periodCloseRitualBeats` — **референс «до»** в колонке **R0**.

| ID | Идея | Плюсы | Минусы |
|----|------|-------|--------|
| **R0** | Символы в круге (как prod) | — | Не совпадает с FinanceMetricIcons |
| **A** | **SVG 20px** в круге: `IconMetricTrendUp/Down`, wallet, shield — tone border | Единый язык | 4–5 новых mapping в util |
| **B** | **Badge 28px** без круга: только иконка + tone color | Компактнее ritual | Меньше «удар» |
| **C** | A + **лёгкий glow** на pos/neg (как juice, без confetti) | Game feel | Риск перегруза |

**Рекомендация:** A или C; R0 не переносить в ★.

# Asset cards — design lab

Статичная витрина вариантов карточек активов (без API и React). Для сравнения перед внедрением в `CapitalPortfolioPanels`.

## Открыть локально

```bash
cd design-lab/asset-cards
npx serve .
```

Или откройте `index.html` в браузере (часть шрифтов подтянется с Google Fonts).

## Что внутри

| ID | Название | Идея |
|----|----------|------|
| **ref** | Сейчас (текст) | Базовая строка «Стоимость · Обслуживание · Доход» |
| **A** | Иконки в рамках | Монеты / ↓ красный / ↑ зелёный — ваш референс |
| **B** | Компактная полоса | Те же иконки, плотнее для списка шаблонов |
| **C** | Строки метрик | Иконка в рамке + подпись + число по строкам |
| **D** | Крупная стоимость | Главная цифра + два chip снизу |
| **E** | Цветная полоса | Акцент слева + сетка метрик |
| **F** | Плитки 2×2 | Четыре ячейки (стоимость, обслуж., доход, чистый поток) |

### Серия 2 (по мотивам E, без блоков и подписей)

| ID | Название | Идея |
|----|----------|------|
| **G ★** | Полоса + линия иконок | Целевой референс: рамки xs, только числа, tooltip |
| **H** | Без рамок | Только цветные иконки + числа — **выбран для prod** (`AssetTemplateMetrics`) |
| **I** | Строки как C | Вертикально, без текстовых labels |
| **J** | Линия с «·» | Компактные разделители между метриками |
| **K** | space-between | Метрики на всю ширину карточки |

Стрелки: **красная ↓** — расход/обслуживание, **зелёная ↑** — доход (явные вертикальные стрелки).

Переключатель **Светлая / Тёмная** имитирует фон TMA. Подсказки: наведение или Tab + focus на метрике.

# Иллюстрации сценариев — 4 набора (как «Режим игры»)

Стиль-референс: [`ModeIllustrations.jsx`](../../../frontend-react/src/components/mqx/icons/ModeIllustrations.jsx) — `viewBox="0 0 72 72"`, штрих + мягкие заливки, орбита из пунктира.

Размер на шаблонах (крупнее save-kind): подложка **56×56**, иллюстрация **40×40** (режим: 48 / 36).

| ID | Идея | Читаемость |
|----|------|------------|
| **A — I-Scene** ★ | Мини-сцена из 2–3 символов + орбита (паритет «Игра») | Максимум смысла |
| **B — I-Focus** | Один крупный символ в мягком круге | Самый чистый |
| **C — I-Duo** | Два равных глифа без орбиты | Баланс простоты и детали |
| **D — I-Bold** | Толстый контур 2.25px, меньше полупрозрачности | Жёстче, TMA на солнце |

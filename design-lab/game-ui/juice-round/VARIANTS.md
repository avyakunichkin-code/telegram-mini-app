# Juice round — варианты A–D

**Раунд:** `design-lab/game-ui/juice-round/`  
**Тон:** bold (см. brainstorm-сессию)  
**Не в scope раунда:** смена компоновки S5, tab-modes (волна 2).

---

## A — Gain (зарплата)

**Идея:** действие = маленькая награда, не бухгалтерская проводка.

| Элемент | Поведение |
|---------|-----------|
| Chip | pop + emerald ring |
| Flyout | `+12 400 ₽` вверх, fade |
| Баланс | tabular-скачок, класс `--pos` |
| Toast | снизу: «Зарплата в кошелёк — ход стал сильнее» |

**Prod:** `claim-salary`, опционально пополнение подушки.

---

## B — Risk (просрочка)

**Идея:** боль ошибки ощущается телом UI, не только красной цифрой.

| Элемент | Поведение |
|---------|-----------|
| Row | shake + border danger |
| Монетка | задержка 200ms, bubble с **последствием** |

**Prod:** highlight просрочки в финансах, tail периода.

---

## C — Turn ritual (ход завершён)

**Идея:** закрытие месяца = **сцена**, не spreadsheet.

| Элемент | Поведение |
|---------|-----------|
| Overlay | violet full-screen |
| Заголовок | «Ход завершён!» + период |
| Beats | 3 строки истории (stagger) |
| CTA | «Дальше» |

**Prod:** заменить/дополнить compact sheet из `period-close` lab.

---

## D — Warning (зарплата сгорит)

**Идея:** модалка с персонажем и ставкой, не системный alert.

| Элемент | Поведение |
|---------|-----------|
| Scrim + modal | slide-up |
| Kicker | amber «Перед ходом» |
| Copy | **12 400 ₽** не придут — explicit loss |
| Actions | primary «Сначала зарплата» / danger «Всё равно закрыть» |

**Prod:** существующий guard перед `advancePeriod`.

---

## Общие правила

- `prefers-reduced-motion`: анимации отключаются, смысл остаётся (классы, текст).
- Длительность сцен ≤ 1 s (кроме C, удерживается до «Дальше»).
- Русский, **ты**, без EN.

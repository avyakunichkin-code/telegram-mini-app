# Варианты: dashboard-needs-round v4

## Исправления v4

- **Сворачивание/раскрытие:** переключение по шапке `».mqx-needs-block__header«` (классы `.is-expanded` + `aria-expanded`).
- **Шапка (свернуто):** маскот слева; справа в 1 строке заголовок `Потребности` + ссылка `как улучшить →` (без вопроса и без chevron).
- **2 строка (свернуто):** слева слабый индикатор; затем шкала **4px** (градиент dark→light по зоне); справа статус (`Истощение/Норма/...`).
- **Шапка (раскрыто):** подсказка `?` показывается рядом с заголовком (hint скрывается).
- **Шкала:** высота **4px**, градиент dark→light по зоне.
- **Кнопка «Улучшить»:** только в раскрытии, в стиле chips (более компактная, без рамки/обводки).
- **Вертикальные столбцы:** не используются (только горизонтальные шкалы).

## Основной макет ★

- Bleed risk banner
- Grid: маскот слева (увеличен) | заголовок/акции справа
- Collapsed: 1 горизонтальная мини-шкала min-оси
- Expanded: 4 горизонтальные шкалы + chip «Улучшить» в раскрытии

## Запуск

```powershell
.\sync-lab.sh
npx serve .
```

Hard refresh (Ctrl+F5) после обновления.

# Row actions — кнопки + / удаление в списках

Этап 1 MQX: варианты до React в prod.

## Утверждения

| Дата | Решение |
|------|---------|
| 2026-05-19 | **B** — компактный + / удаление, glass, hover на desktop, hit-area 44px |
| 2026-05-19 | **E2** — в строке позиции **не дублировать** суммы в `subtitle`, если ниже уже есть метрики с иконками (`MetricsRow` / `*PositionMetrics`) |
| 2026-05-19 | **F2** — в кнопке удаления по умолчанию **иконка корзины** (`IconMetricTrash`) → prod |

Подтверждение **для всех** разрушающих действий — без изменений.

### Метрики в той же строке (согласовано 2026-05)

- Порядок слева направо: **сумма (coin)** → **платежи, красные (down)** → **доход, зелёный (up)** → **ставка % только числом** (глиф `percent`, красный = платим, зелёный = получаем, **без** суффикса `%` в значении) → **страховки / срок (term)** и т.д.
- Суммы в ₽ **без** `/мес` в подписи; пояснение «за период» / помесячная модель — в подсказке (`tip` / `title`).

Plan (`BaseParamsScreen`, `PlanExpenseEditor`) — **вне scope** этой волны.

## Иконки

- В полоске метрик — только утверждённый набор из [`FinanceMetricIcons.jsx`](../../frontend-react/src/components/mqx/icons/FinanceMetricIcons.jsx) через `MetricInlineItem` (`coin`, `down`, `up`, `percent`, `term`).
- **`IconMetricTrash`** — кнопка удаления по умолчанию (F2).

## Тёмная тема (lab)

В `[data-theme='dark']` заданы `--lab-text` / `--lab-text-soft` — body и заголовки читаются на `--mq-mist`. Проверка: переключатель в шапке `index.html`.

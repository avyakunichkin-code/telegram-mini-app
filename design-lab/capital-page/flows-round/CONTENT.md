# Контент: страница «Управление капиталом» (6 аккордеонов)

## Структура

```
[ Hero ]
[ details: Доходы ]
[ details: Расходы ]
[ details: Инвестиции ]
[ details: Страховки ]
[ details: Имущество ]
[ details: Обязательства ]
```

Табы **не используются**. Каждый раздел — свой accent снизу summary.

## Типы контента

| Раздел | Тело аккордеона |
|--------|----------------|
| Доходы / Расходы | Категории + строки (`buildCapitalPeriodFlows.js`) |
| Инвестиции … Обязательства | `cap-lead` + сегмент «Оформить/Добавить \| Мои (N)» + каталог/позиции |

## Prod

# Контент: Доходы и Расходы (управление капиталом)

## Позиция на странице

```
[ Hero: Управление капиталом ]
[ ★ Доходы | Расходы — первая строка ]
[ Табы: Инвестиции · Бюджет · Страховки · Имущество · Обязательства ]
[ Контент активного раздела ]
```

Блок «Бюджет» во вкладках остаётся для **редактирования** статей жизни; сводка «На жизнь» дублируется в **Расходах** как цифра периода.

## Доходы (статьи)

| Статья | Источник | Разбивка |
|--------|----------|----------|
| Зарплата | `overview.salary.monthly_amount` | одна строка |
| Инвестиции | активные `investPositions` | по `kind`: Депозиты, Облигации; внутри — позиции (опционально в prod) |
| Доходное имущество | `overview.assets` где `monthly_income > 0` | по `kind` (Аренда, Авто, …) |

**Итого доходов** = сумма трёх статей (в MVP зарплата + доход активов; инвестиции — начисление 1/12 ставки, не в `total_monthly_income` API).

## Расходы (статьи)

| Статья | Источник | Разбивка |
|--------|----------|----------|
| На жизнь | `monthly_burn_total` / `monthly_burn_breakdown.lines` | категории бюджета |
| Содержание имущества | `assets[].monthly_maintenance_cost` | по `kind` |
| Платежи по обязательствам | `liabilities[].monthly_payment` | по названию позиции |
| Страховки | `policies[].monthly_premium` | по полису / продукту |

**Итого расходов** = сумма статей (≈ `total_monthly_outflow` + страховки, если премии не в overview).

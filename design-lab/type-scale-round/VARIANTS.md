# Type scale round

| Решение | Значение |
|---------|----------|
| 13px суммы/вторичный | → **12px** `--mq-fs-caption` |
| 9px kicker chip | → **10px** `--mq-fs-micro` |
| Hero display | **A 26px** vs **B 28px** — выбор в lab §3 |

## Целевая шкала

| Токен | px |
|-------|-----|
| `--mq-fs-display` | 26 или 28 (после выбора) |
| `--mq-fs-title` | 20 |
| `--mq-fs-heading` | 14 |
| `--mq-fs-body` | 15 |
| `--mq-fs-caption` | 12 |
| `--mq-fs-small` | 11 |
| `--mq-fs-micro` | 10 |

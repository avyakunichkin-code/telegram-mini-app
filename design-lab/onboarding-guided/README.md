## Шаги (кратко)

1. Период / «Месяц открыт» (TB1, без секундомера) → **~10 с** практика  
2. **Зарплата** → ждём клик  
3. **Пополнить** подушку → ждём любое пополнение  
4. **Закрыть месяц** → **~10 с** практика (после 3)  
5. Прощание → **Начать игру**

## Skip

1-й «Пропустить» = шаг; 2-й = весь онбординг.

## Когда показывать

Только **первая игра** после выбора **Game Mode** (автостарт с **самым простым** шаблоном). Зарплата 0 в шаблоне — **баг**, не сценарий онбординга.

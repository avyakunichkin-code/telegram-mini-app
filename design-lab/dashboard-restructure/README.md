# Dashboard — уровень-аккордеон + подушка (ползунок)

**Задача:** «Уровень» — сворачиваемый блок с **целями победы внутри**; «Дашборд периода» — только финансы (без целей); пополнение/снятие подушки — сумма как в **Инвестициях** (`InvestAmountControl`).

Связанный архив компоновок: [`../dashboard-dual-accordion/`](../dashboard-dual-accordion/) (D1–D5, шире по IA).

## Запуск

```bash
cd design-lab/dashboard-restructure
npx serve .
```

Тема — переключатель в шапке. Клик по заголовку аккордеона — раскрытие. В блоке **S** — кнопки «В подушку» / «Снять» открывают модалку в макете.

## Варианты структуры (R)

| ID | Идея | Уровень (свёрнут) | В развороте уровня | Дашборд периода |
|----|------|-------------------|--------------------|-----------------|
| **R1** | Канон prod+ | XP + **Победа 2/3** + очки | Цели v2 + бары доход/долг | Chips → только **Финансы**; целей нет |
| **R2** | Компакт | Одна строка + XP-bar | Цели + бары | Как R1 |
| **R3** | Превью целей | XP + 3 чипа целей (✓✓✗) | Полный список целей + бары | Как R1; подсказка «только финансы» |

**Действия периода** во всех R — **снаружи** аккордеонов (как текущий prod).

## Варианты подушки (S)

| ID | Идея | Отличие от invest-forms |
|----|------|-------------------------|
| **S1** | Паритет | Тот же паттерн: подчёркнутый ввод + фиолетовый ползунок + «На счёте» / «В подушке» |
| **S2** | Быстрые доли | Чипы **25% · 50% · Всё** над ползунком; акцент **emerald** (подушка) |
| **S3** | Sheet | Нижний лист: крупная сумма, ползунок, балансы cash/подушка внизу |

## Связь с prod

## Утверждение

Примеры: «**R1** + **S2**» или «**R3** свёрнутый превью, подушка **S1**».

До явного «утверждаем …» — только design-lab, без правок `DashboardPremium.jsx`.

## 📖 О проекте

**Money Quest** — это Telegram Mini App, который в игровой форме обучает основам финансовой грамотности. Игрок управляет личным бюджетом, инвестирует, погашает долги и накапливает капитал в условиях игровых периодов.

### 🎯 Ключевые особенности

- ✅ **Полноценная JWT-авторизация** — регистрация и вход через логин/пароль
- ✅ **Игровые периоды** — каждый период требует выполнения обязательных действий (можно забыть → будут последствия)
- ✅ **Система времени** — Pause/Play/Next с реальным таймером
- ✅ **Финансовый скоринг** — оценка финансового здоровья (0-100)
- ✅ **Прогрессия и уровни** — XP за выполнение целей
- ✅ **Подушка безопасности** — накопления с возможностью откладывать
- ✅ **События с выбором (easy)** — решения дают последствия (доходы/расходы/переводы в подушку)
- ✅ **Инвестиции (easy MVP)** — депозиты (капитализация) и облигации (купоны)
- ✅ **Страховки (easy MVP)** — полисы с ежемесячными премиями
- ✅ **Типовые активы (easy)** — шаблоны активов (квартира/аренда/авто/такси) с обслуживанием и доходом
- ✅ **Real-time обновления** — счетчик времени обновляется локально и синхронизируется с сервером
- ✅ **Telegram Web App API** — автоматическая адаптация под тему и платформу

## 🧱 Архитектура

### Бэкенд
- `backend/main.py` — FastAPI + автосоздание таблиц и “лёгкая автомиграция” колонок
- `backend/app/models.py` — SQLAlchemy модели (профили, финансы, события, инвестиции, страховки)
- `backend/app/game_period.py` — логика конца периода (списания/доходы/просрочки/начисления)

### Фронтенд
- `frontend-react/src/hooks/useGame.js` — состояние игры, таймер, синхронизация, pending события
- `frontend-react/src/components/GameScreen.jsx` — основной экран + модалка событий
- `frontend-react/src/components/FinanceSection.jsx` — инвестиции/страховки/шаблоны активов

## 🚀 Деплой

### Бэкенд (Render)

1. **Создайте PostgreSQL базу** на Render (бесплатный тариф)
2. **Создайте Web Service** из GitHub репозитория
3. **Добавьте переменные окружения**:
    - `DATABASE_URL` — Internal URL из PostgreSQL
    - `SECRET_KEY` — любая сложная строка
4. **Команды**:
    - Build: `pip install -r requirements.txt`
    - Start: `uvicorn main:app --host 0.0.0.0 --port 10000`

## 🎮 Игровая механика

### Игровые периоды
- Каждый период = один месяц игрового времени
- Игрок должен успеть выполнить обязательные действия
- Время можно ускорять, ставить на паузу или переходить вручную

### Действия периода
1. **Получить зарплату** — один раз за период. Не нажал кнопку → зарплата за период **сгорает**
2. **Отложить в подушку безопасности** — любой суммой из доступного дохода

### Просрочки обязательств (MVP)
- Платежи по обязательствам пытаются списаться в конце периода
- Если денег не хватает, остаток уходит в **просрочку** (пока без штрафов/пени)

### События (easy)
- В некоторых периодах появляется событие с выбором (2–3 варианта)
- Эффекты (MVP): изменение баланса / перевод в/из подушки

### Инвестиции (easy MVP)
- **Депозит**: проценты капитализируются в сумму депозита
- **Облигации**: купоны начисляются на баланс

### Страховки (easy MVP)
- Полисы списывают ежемесячную премию в конце периода

### Финансовый скоринг
- **80-100** — Финансовый стратег
- **55-79** — Уверенный планировщик
- **30-54** — Начинающий инвестор
- **0-29** — Финансовый новичок

### Прогрессия
- За выполнение действий начисляется XP
- При достижении 100 XP — повышение уровня
- Уровень влияет на игровые бонусы

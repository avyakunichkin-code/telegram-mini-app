# `src/components/` — legacy quarantine

Этот каталог — **карантин legacy**. Он существует для поддержки текущего приложения, но **не должен расти** новыми UI‑паттернами.

## Что можно и что нельзя

- **Нельзя** добавлять новые видимые UI‑паттерны напрямую в `components/`.
  - Любой новый/существенно меняющий вид паттерн — только через **MQX** по циклу:
    `design-lab/ → утверждение → components/mqx/ + #/dev/mqx → внедрение`
    (см. `src/components/mqx/DESIGN_WORKFLOW.md`).

- **Можно** делать точечные правки legacy, если это:
  - багфикс/hotfix без смены дизайна;
  - “touch-it move-it” перенос по касанию (вынос в `screens/…` или MQX, оставляя thin-wrapper).

## Явный карантин

Компоненты вида:
- `*Premium.jsx`
- `*Section.jsx`

считаются legacy‑контейнерами и **не должны импортироваться из `src/screens/**`**.
